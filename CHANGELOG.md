# Changelog

All notable changes to this ML portfolio website will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.1.0] - 2025-09-08

### Added
- Enhanced WSL compatibility with polling-based file watching
- Background API server for projects and skills data
- Improved development workflow with automated server restart script

### Changed
- Improved Vite configuration for better HMR in WSL environments
- Updated development workflow for better cross-platform compatibility

### Fixed
- Hot Module Replacement (HMR) issues in WSL environment
- Multiple conflicting development servers causing file watching problems
- TypeScript compilation errors and build optimization

### Development Experience
- Added `dev-restart.sh` script for clean development server restarts
- Improved error logging and development diagnostics
- Better development server management and port conflict resolution

## [1.0.0] - 2025-09-07

### Added
- Initial portfolio website structure
- Modern, minimalist design with mobile-first approach
- Hero section with ML developer value proposition
- About section highlighting career transition story
- Projects showcase featuring 4 ML projects
- Skills section with technical ML competencies
- Contact section optimized for startup recruiters
- Responsive design with smooth CSS transitions
- Accessibility features and performance optimizations
- Clean, semantic HTML structure
- Professional color palette with accent color
- Typography optimized for mobile readability

### Technical Features
- Mobile-first responsive design
- Fast loading times
- Semantic HTML5 structure
- CSS Grid and Flexbox layouts
- Smooth CSS animations
- ARIA accessibility labels
- Performance optimized assets
